# Webfonts

`CommitMonoNF-*.woff2` are subsets of **CommitMono Nerd Font** (CommitMono by
Eigil Nikolajsen, patched by the Nerd Fonts project), licensed under the
[SIL Open Font License 1.1](https://github.com/eigilnikolajsen/commit-mono/blob/main/LICENSE).

Subset with `fontTools.subset` to: Basic Latin, Latin-1, General Punctuation,
Arrows (U+2190–21FF), Box Drawing (U+2500–257F), Geometric Shapes (U+25A0–25FF),
and Powerline glyphs (U+E0A0–E0D7). If a page needs glyphs outside those ranges
(e.g. Nerd Font dev icons), re-subset from the full font with wider `--unicodes`.
